## Executive Analytics Dashboard

![Healthcare Claims Dashboard](dashboards/healthcare_claims_dashboard.png)

This executive dashboard is built on top of Gold-layer Delta tables using Databricks SQL and provides a business-level view of healthcare claims data.

**Key insights include:**
- Monthly claim amount trends for cost monitoring
- Fraud flag distribution highlighting high-risk claims
- Data quality and reconciliation metrics ensuring pipeline reliability
- Executive KPIs such as Total Claims, Total Claim Amount, and Fraud Rate

The dashboard enables fast, reliable analytics with full traceability back to the Bronze and Silver layers of the Databricks Lakehouse.
